This is a Riemann problem test that corresponds to testcase 5.3 of Titarev et al. 2006: Sonic point test problem. The material obeys the isotropic hyperelastic equation of state due to Romenski (Titarev, Romenski, Toro 2008). Material constants are chosen to correspond to copper, given by ρ0=8.93, α=1, β=3, γ=2, b0=2.1, c0=4.6.

The initial conditions are:

  • F =
    0.800
    010
    001
    , u = (0,0,0), S = 0 if x < 0.55
  • F = I, u = (3,0,0), S = 0 if x > 0.55
and the time to which the simulation is run is t=0.04.

There are three sets of tests run: one set with varying resolution of 100, 200, 400, and 800 cells, and no refinement, and the second set with a fixed effective resolution of 800 cells, with coarsest grid level resolutions 100, 200, and 400, with refinements of factor 2 to achieve the effective resolution.

The third set of tests varies the slope-limiter used for comparison of their various properties. We also choose whether to limit on primitive or conserved variables (for SLIC and MUSCL-Hancock only).

The first two tests are run with five different flux methods, the first-order FORCE and Godunov and the second-order SLIC, MUSCL-Hancock and WAF. The third test is only run for SLIC, MUSCL-Hancock and WAF being the only methods that use limiters.
